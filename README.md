# 남구 · 부산광역시

> 남구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-busan-nam.github.io/**

부산 남구는 오륙도가 바다 위에 우뚝 솟은 부산의 동남단 해안 자치구로, 국립해양조사원이 이곳 오륙도를 기준으로 동해와 남해를 가르는 상징적 경계를 이룹니다. 6·25전쟁에 참전해 산화한 세계 각국 유엔군이 잠든 세계 유일의 유엔기념공원(재한유엔기념공원)이 대연동에 자리해 'UN평화문화특구'로 지정된 평화의 도시이기도 합니다. 이기대와 신선대로 이어지는 해안 절벽과 파식대지가 갈맷길·해파랑길의 출발 구간을 이루어 걷기 여행의 명소로 사랑받습니다. 문현동 일대에는 부산국제금융센터(BIFC)와 문현금융단지가 들어서 동남권 해양·파생 금융의 중심축을 담당합니다. 경성대·부경대·동명대가 밀집한 대학가와 감만·신선대 컨테이너 부두가 공존하여 젊음과 항만 물류가 어우러진 독특한 도시색을 지닙니다. 바다·평화·금융·청년 문화가 한데 모인 부산 남구는 자연 경관과 도시 기능이 조화를 이룬 지역입니다.
